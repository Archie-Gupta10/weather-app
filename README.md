  # Weather App                                                                                                                                                                                                                          
                                                                                                                                                                                                                                           
    Weather App is a clean, responsive, and interactive web application that provides real-time weather information and multi-day forecasts for cities worldwide.                                                                          
                                                                                                                                                                                                                                           
    ## Features                                                                                                                                                                                                                            
                                                                                                                                                                                                                                           
    * **Real-Time Weather Metrics:** Instantly retrieve current temperature, feels-like temperature, humidity, wind speed, atmospheric pressure, and general conditions.                                                                   
    * **Global Location Search:** Search for any city or region globally using a simple search input field.                                                                                                                                
    * **Geographical Auto-Detection:** Option to fetch local weather conditions automatically using the browser geolocation API.                                                                                                           
    * **Forecast Display:** Multi-day and hourly weather forecasting to help users plan ahead.                                                                                                                                             
    * **Dynamic Backgrounds:** The interface dynamically changes colors or background themes based on the retrieved weather conditions (such as sunny, rainy, cloudy, or snowy).                                                           
    * **Responsive Layout:** Designed to scale and display beautifully across mobile phones, tablets, and desktop displays.                                                                                                                
                                                                                                                                                                                                                                           
    ## Technologies Used                                                                                                                                                                                                                   
                                                                                                                                                                                                                                           
    * **Structure:** HTML5                                                                                                                                                                                                                 
    * **Styling:** CSS3 (Custom properties, grid, flexbox, and transitions)                                                                                                                                                                
    * **Logic:** Vanilla ES6+ JavaScript (Fetch API for asynchronous data loading)                                                                                                                                                         
    * **Data Provider:** Integration with public weather data APIs (such as OpenWeatherMap or WeatherAPI)                                                                                                                                  
                                                                                                                                                                                                                                           
    ## Getting Started                                                                                                                                                                                                                     
                                                                                                                                                                                                                                           
    Follow these steps to set up and run the project locally.                                                                                                                                                                              
                                                                                                                                                                                                                                           
    ### Prerequisites

       ```bash                                                                                                                                                                                                                             
       git clone https://github.com/Archie-Gupta10/weather-app.git                                                                                                                                                                         
                                                                                                                                                                                                                                           
  2. Navigate to the project directory:                                                                                                                                                                                                    
    cd weather-app

    const API_KEY = "YOUR_API_KEY_HERE";
